
# Image

Use the `<Image />` component to render a primitive image that saves the aspect ratio.

```.jsx
<Image
  alt='Brooklyn'
  src='https://source.unsplash.com/bITjK6W2Alw/1024x768?q=20'
/>
```

```.jsx
<Image
  alt='Brooklyn'
  src='https://source.unsplash.com/bITjK6W2Alw/1024x768?q=20'
  width='300'
  height='200'
/>
```
